🔥 ربط Firebase للمزامنة بين الفنيين

لتفعيل المزامنة الفورية بين جميع الأجهزة، اتبع هذه الخطوات مرة واحدة فقط:

1
اذهب إلى console.firebase.google.com وأنشئ مشروعاً مجانياً
2
فعّل Realtime Database واختر Start in test mode
3
انقر ⚙️ أعلى يسار، افتح إعدادات المشروع، انسخ firebaseConfig والصقه هنا
4
أرسل نفس ملف HTML لكل الفنيين — البيانات ستُزامن لحظياً بين الكل
